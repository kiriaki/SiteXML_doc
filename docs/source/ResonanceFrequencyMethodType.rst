ResonanceFrequencyMethodType
============================

**Type:** ``simpleType``
**Description:**
  Method by which resonance frequency (dominant_frequency) has been determined.
  				Used to determine the Quality Index of the provided f0 of a site.
  				For more information, see SERA Deliverable 7.1, Appendix II.

Possible Values
--------------

* ``HVSR EARTHQUAKE RECORDS`` – * ``HVSR NOISE`` – * ``SSR EARTHQUAKE RECORDS`` – * ``SSR NOISE`` – * ``INFERRED`` –
